# 第一阶段·第一层：核心定位（必答，3问）

> 本层问题为必答项，确定小说的基本骨架。每个问题提供热门选项 + 自由输入。

---

## Q1：题材与创意概要

使用 `AskUserQuestion` 工具询问：

```
Question: 你想要创作什么题材的小说？
Options:
- 悬疑推理（侦探、破案、解谜）
- 言情（现代都市 / 古代宫廷 / 穿越时空）
- 奇幻玄幻（魔法、异世界、修真）
- 科幻未来（科技、太空、末世）
- 武侠仙侠 / 历史架空（江湖、门派、朝堂权谋）
- 都市现实 / 成长（生活、职场、社会）
- ⭐自由描述（我有明确的想法，让我自己说）
```

> **偏好集成**：如果 `favoriteGenres` 中有匹配项，将对应选项排在最前面并标记⭐。

**记录**：`题材 = [用户选择]`
**记录**：`创意概要 = [用户在"自由描述"中已包含足够信息]`

## Q3：核心冲突与驱动力

使用 `AskUserQuestion` 工具询问：

```
Question: 小说的核心冲突是什么？
Options:
- 生死存亡（生存危机、逃出生天）
- 查明真相（寻找答案、揭露秘密）
- 爱情阻碍（追求真爱、克服阻碍）
- 复仇雪恨（复仇计划、伸张正义）
- 权力争夺（竞争上位、资源争夺）
- 成长突破（自我突破、实现价值）
- 守护保护（守护重要的人或事）
- 自由描述
```

**记录**：`核心冲突 = [用户选择]`

**追问 — 驱动力**：
```
Question: 是什么推动主角不断向前？主角的内在驱动力是什么？
Options:
- 复仇（为某人/某事而战）
- 爱情（为所爱之人）
- 责任/使命（不得不做）
- 好奇心/求知欲（想知道真相）
- 生存（活下去）
- 野心（想要得到某样东西）
- 自由描述
```

**记录**：`驱动力 = [用户回答]`

**追问 — 核心循环（Core Loop）**：
```
Question: 主角会反复做什么来获得收益并进入更难挑战？
Options:
- 战斗升级：进入危险区域 → 击败强敌 → 获得经验/装备 → 升级 → 去更难区域
- 破案推进：接触异常 → 找线索 → 识破误导 → 获得真相/权限 → 进入更深案件
- 经营扩张：发现需求 → 调动资源 → 完成交易/产品 → 获得资金人脉 → 进入更大市场
- 职场逆袭：遭遇不公 → 用专业能力破局 → 获得筹码 → 升级话语权 → 面对更高层博弈
- 情感推进：误会/阻碍 → 主角选择 → 关系变化 → 情感兑现 → 更深矛盾出现
- 自由描述
```

**记录**：`核心循环 = [用户回答]`

```markdown
🧩 **优势来源**：[主角熟悉领域/优势来源]
💥 **冲突**：[核心冲突] | 驱动力：[驱动力]
🔁 **核心循环**：[核心循环]
```

将第一层的回答同步到 `user-preferences.json`（不打扰用户）。

偏好数据结构与更新规则详见 → [shared-infrastructure.md](shared-infrastructure.md)

→ 进入 **第二层：深度定制与创作规格（Q4-Q8）**，详见 [phase1-layer2-customize.md](phase1-layer2-customize.md)
