# 黄金三章指南

黄金三章指小说开头三章必须尽快抓住读者。前三章不是普通开篇，而是全书的第一道留存门槛：要让读者认识主角、理解故事路线、感到悬念和期待，并在第三章看到一次小高潮。

## 总原则

1. **聚焦主角**：前三章减少无关人物，镜头尽量跟随主角或主角关系最密切的人。
2. **开篇明志**：前三章必须明确题材、基调、叙事路线和核心爽点。
3. **简洁明快**：删掉无关设定、无关闲聊、过早展开的大世界说明。
4. **挑起悬念**：每章都要让读者想看下一章，第三章要把期待感推高。
5. **低级错误归零**：人名、时间、视角、设定、动机不能混乱。

## 主角共鸣九特性

前三章至少选择 1-2 个特性集中呈现，不要平均铺满。

| 特性 | 用法 |
|---|---|
| 勇敢 | 让主角在压力下仍做出选择 |
| 不公平的伤害 | 让读者觉得主角不该被这样对待 |
| 本事 | 展示主角确实有一手，哪怕暂时被压制 |
| 幽默 | 用反应和台词建立亲近感 |
| 善良 | 让主角在小事上显出底色 |
| 身处险境 | 直接把主角放进危险局面 |
| 受到朋友和家人的深爱 | 用关系承托主角，不靠旁白夸 |
| 努力 | 展示他为目标付出的具体行动 |
| 执着 | 让主角在失败后仍不松手 |

## 三章结构

### 第1章：启示

目标：构建主角、现状和未来期待。

必须完成：
- 给主角一个可共鸣特性，并用动作或场景展示出来。
- 描写主角目前的生活状态，但不要写成流水账。
- 埋下一根“长钉子”：一个能在第2章引爆的问题、秘密、关系或危险。
- 给出未来展望：主角自己的愿望，或亲友对主角的期待。
- 结尾追读理由指向第2章的转折。

避免：
- 大段世界观设定。
- 一口气扔出大量人物。
- 主角缺席或主角目标不清。

### 第2章：转折

目标：让主角狠狠受挫，让故事无法回到原状。

必须完成：
- 引爆第1章埋下的长钉子。
- 强化主角的不公平处境、险境、梦想受阻或关系受伤。
- 让主角面临重大损失：生命、梦想、亲友、尊严、自由、身份或最在乎的东西。
- 给主角一个不得不行动的理由。
- 结尾把压力推到第三章的小高潮。

避免：
- 转折太轻，只是小麻烦。
- 主角被动挨打但没有选择。
- 用巧合硬推剧情。

### 第3章：小高潮

目标：给读者第一次兑现，但只兑现一部分。

必须完成：
- 让主角在压力下主动出手。
- 形成一个小高潮：对抗、揭露、反杀、逃脱、证明自己、心理打脸等。
- 只给有限胜利，不能让主角一下无敌。
- 让看不起主角的人或阻碍主角的力量丢一次面子。
- 画出更大的饼：读者看见未来爽点，但还吃不到。
- 结尾打开第一卷或后续主线。

避免：
- 第三章仍然只铺垫。
- 爽点过满，直接解决核心矛盾。
- 小高潮和主线无关。

## QA 专项门禁

前三章 QA 必须增加“黄金三章专项”：

| 章节 | 必查项 | 阻塞条件 |
|---|---|---|
| 第1章 | 主角共鸣、现状、长钉子、未来展望、风格路线 | 主角目标不清或没有长钉子 |
| 第2章 | 长钉子引爆、重大损失、不得不行动 | 转折太轻或主角没有行动理由 |
| 第3章 | 主动出手、小高潮、有限胜利、更大期待 | 没有小高潮或直接解决主线 |

如果前三章任一章未通过黄金三章专项，即使总分达到 85，也不得标记 `completed`。
